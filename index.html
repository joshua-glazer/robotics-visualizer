How to make an ENCS website like this one:

Software used: Mobaxterm
Why: You can directly manipulate the files and folders with a GUI rather than just a shell.

Steps:

  1. Open a session in Mobaxterm and choose SSH
  2. Sign in using login.encs.concordia.ca (default port 22 works)
  3. It will ask you for your username, then your password. use your ENCS username and password
  4. On the left in the file explorer thing switch from /nfs/home/username/ to /www/home/username/
  5. Now you can replace whatever was in that directory with your own HTML files etc. to make a website!
  6. Last tip is to make sure the permissions on any file you add allow Other people to Read it
